# Airtable URL CLI - Created by @dotangad and modified by me

## Setup
1. Clone
```sh
git clone git@github.com:kavin25/airtable-url-cli
cd airtable-url-cli
```

2. Install
```sh
npm install
```

3. Setup env
```sh
cp .env.example .env && nvim .env
```

4. Global Install CLI
```sh
npm install -g
```

## Usage
```sh
url -h
```
> Help Command which gives you usage instructions
